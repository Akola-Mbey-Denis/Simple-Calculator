# Simple-Calculator
This is a simple calculator built using HTML,CSS and JavaScript. 
Limitations:
The left and right braces logic was not implemented.
Thus such evaluations donot work.
The following operations can be performed:
Addition (+)
Subtraction (-)
Division (/)
Multiplication (*)
Modulus (Mod)
Percentage (%)
toggle (+/-)
square (sq) 
squareroot(sqrt)
clear (clr)
Enjoy

Tools used:
JavaScript
HTML
CSS
